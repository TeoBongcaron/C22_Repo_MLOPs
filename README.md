# Advanced ML CI/CD Pipeline  
A production‑grade MLOps project demonstrating containerized ML inference, automated testing, and continuous integration/continuous delivery using GitHub Actions.

This project serves a pre‑trained ONNX sentiment analysis model through a FastAPI web service. The repository includes functional tests, Docker packaging, and a full CI/CD workflow that builds, tests, and pushes the container image to Docker Hub.

---

## Features

- **FastAPI inference service** for real‑time sentiment analysis  
- **ONNX Runtime** for fast, portable model execution  
- **Containerized deployment** using Docker  
- **Automated CI/CD pipeline** with GitHub Actions  
- **Functional API tests** using pytest  
- **Health check endpoint** for monitoring  
- **Production‑ready Uvicorn server**
